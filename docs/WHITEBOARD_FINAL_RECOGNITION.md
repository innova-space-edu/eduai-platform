# Reconocimiento final del lienzo

El cuaderno de desarrollo debe guardar los trazos al instante y evitar enviar OCR en cada trazo.

Flujo recomendado:

1. Mientras el estudiante escribe, se guardan los trazos del lienzo.
2. El botón manual permite generar LaTeX si el docente o estudiante quiere revisar en pantalla.
3. Al cambiar de pregunta o entregar, se intenta generar el LaTeX final desde el lienzo guardado.
4. Si el proveedor externo está dormido o tarda, el lienzo igualmente queda guardado como evidencia.
