## ¿Qué es EduAI Platform?

EduAI Platform es una plataforma educativa de siguiente generación que combina **18+ agentes de inteligencia artificial especializados** y un **Creator Studio con 8 formatos de salida** para ofrecer una experiencia de aprendizaje completamente personalizada, adaptativa y gamificada. Va más allá de un simple chatbot — es un ecosistema inteligente donde múltiples agentes colaboran para que el estudiante aprenda más rápido y de forma más profunda.

Diseñada especialmente para el contexto educativo chileno, incluye cobertura curricular completa del **MINEDUC** (Parvularia, Básica y Media).

## Funcionalidades principales

### Para estudiantes
- **Sesiones de estudio sobre cualquier tema** — historia, ciencias, matemáticas, idiomas, y más
- **4 modos de aprendizaje:** Normal, Socrático, Evaluación y Colaborativo
- **Quiz adaptativo** que ajusta dificultad automáticamente según tu desempeño
- **Modo Examen completo** con timer, corrección automática y retroalimentación detallada
- **Chat con Papers PDF** — sube un paper académico y conversa profundamente sobre él (ES/EN y otros idiomas)
- **Visualizaciones automáticas:** imágenes IA, diagramas Mermaid, gráficos Chart.js
- **Matemáticas con LaTeX** — fórmulas renderizadas como en un libro de texto profesional
- **Narración por voz (TTS)** — el contenido se puede escuchar
- **Historial de sesiones** organizado como carpetas con estadísticas
- **Repaso espaciado inteligente** basado en el algoritmo SM-2 (Ebbinghaus)

### ✨ Creator Studio — NUEVO

Un motor universal de creación de material educativo. Transforma cualquier contenido (tema, texto, URL, PDF, DOCX) en 8 formatos diferentes:

| Formato | Descripción | Descargas |
|---------|-------------|-----------|
| 📊 **Infografía** | Visual con secciones, datos clave y estadísticas | PNG, JPG, PDF |
| 📑 **Presentación** | Slides profesionales con notas del orador | PPTX, PDF, PNG |
| 🎨 **Afiche/Poster** | Poster visual con puntos destacados | PNG, JPG, PDF |
| 🎙️ **Podcast** | Diálogo entre 2 hosts con voces Edge TTS reales | MP3, PDF guión, TXT |
| 🧠 **Mapa Mental** | Grafo interactivo de conceptos conectados | PNG, PDF |
| 📇 **Flashcards** | Tarjetas de estudio con niveles de dificultad | PDF, PNG |
| ✅ **Quiz** | Evaluación con opción múltiple, V/F y explicaciones | PDF |
| ⏳ **Timeline** | Línea temporal con eventos y niveles de importancia | PNG, PDF |

**Características del Creator Studio:**
- **Pipeline universal** — una sola API procesa URL (web scraping con Cheerio), PDF, DOCX (Mammoth), texto directo o un tema
- **Procesamiento con Gemini 2.5 Flash** con fallback automático a Groq
- **12 paletas de color** seleccionables que se aplican a la vista previa y a los archivos exportados
- **PDFs estilo Canva** — headers de color, barras laterales, badges numerados, tipografía profesional
- **Presentaciones PPTX reales** generadas con PptxGenJS (descargables, editables en PowerPoint/Google Slides)
- **Podcast con audio MP3 real** — Edge TTS con 2 voces distintas (AlvaroNeural y ElviraNeural), descargable
- **Reproducción en vivo** del podcast con Web Speech API directamente en el navegador
- **Exportación PNG/JPG** de cualquier formato visual con html-to-image a 2x resolución

### 🎙️ Generador de Podcast — NUEVO

Convierte cualquier tema o documento en un podcast educativo estilo NotebookLM/Spotify:

```
Contenido → Gemini genera guión con 25+ segmentos
  → Host A ("El Profesor"): explica con autoridad, datos, analogías
  → Host B ("El Estudiante Curioso"): preguntas inteligentes, reacciones, humor
  → Edge TTS genera MP3 con voces naturales en español
  → Descarga como archivo .mp3
```

- **2 voces distintas**: AlvaroNeural (masculina) y ElviraNeural (femenina)
- **Tono diferenciado**: pitch y velocidad distintos por host
- **Audio profesional**: 24kHz 96kbps mono MP3
- **Server-side**: sin CORS, sin timeout, procesado en tu API

### Para docentes
- **Planificador MINEDUC (APl)** — genera planificaciones de clase alineadas al currículo nacional
- Soporte para todos los niveles: Parvularia, 1° Básico hasta 4° Medio
- Adaptaciones para **Necesidades Educativas Especiales (NEE)**
- Actividades lúdicas, rúbricas, proyectos interdisciplinarios
- Contexto del calendario escolar chileno (Fiestas Patrias, vacaciones de invierno, etc.)

### Colaboración en tiempo real
- **Salas multiusuario** con hasta 10 participantes simultáneos
- **Código de sala único** para invitar compañeros
- **Profesor ACo** — IA moderadora que detecta dudas automáticamente
- Contador real de conectados via Supabase Realtime
- Indicador visual de conexión Realtime activa

### Chat Social tipo Messenger
- **Código de usuario único** de 8 caracteres por persona
- Búsqueda por nombre o código de usuario
- Solicitudes de amistad con aceptar/rechazar
- Mensajes en tiempo real via Supabase Realtime
- **Reacciones** con 6 emojis (aparecen al hover)
- Envío de archivos e imágenes a Supabase Storage
- Estado online/offline con última vez conectado
- Check de leído y notificaciones en tiempo real

### Herramientas generales
- **Investigador** — busca y resume fuentes académicas y papers
- **Redactor** — genera ensayos, informes y cartas formales
- **Matemático** — resolución paso a paso con notación LaTeX profesional
- **Traductor** — traducción multiidioma con explicación lingüística y cultural
- **Generador de Imágenes** — FLUX y Stable Diffusion con galería persistente
- **Visualización automática en estudio** — detección de contenido visual con generación de imagen de apoyo

## Sistema Orquestador de 4 Agentes

Para preguntas complejas, el tutor activa automáticamente un sistema invisible de 4 agentes:

```
Pregunta compleja detectada (>8 palabras o palabras clave)
    ↓
┌─────────────────────────────────────────┐
│  Investigador (Gemini) ─┐               │
│                          ├─ En paralelo │
│  Lógico       (Gemini) ─┘               │
└──────────────┬──────────────────────────┘
               ↓
    Creativo (Gemini) — usa output del investigador
               ↓
    Capitán  (Gemini) — sintetiza todo
               ↓
    Tutor (Groq stream) — responde enriquecido
```

## Instalación y desarrollo

### Requisitos previos
- Node.js 18+
- Cuenta en [Supabase](https://supabase.com) (gratuita)
- API Keys gratuitas (ver abajo)

### 1. Clonar el repositorio
```bash
git clone https://github.com/innova-space-edu/eduai-platform.git
cd eduai-platform
npm install
```

### 2. Configurar variables de entorno
```bash
cp .env.example .env.local
```

```env
# Supabase (supabase.com → gratis)
NEXT_PUBLIC_SUPABASE_URL=https://xxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...

# Groq (console.groq.com → gratis)
GROQ_API_KEY=gsk_xxx...

# Google Gemini (aistudio.google.com → gratis)
GEMINI_API_KEY=AIzaXxx...

# OpenRouter (openrouter.ai → gratis, sin tarjeta)
OPENROUTER_API_KEY=sk-or-xxx...

# HuggingFace (huggingface.co → gratis)
HF_TOKEN_1=hf_xxx...
HF_TOKEN_2=hf_xxx...
HF_TOKEN_3=hf_xxx...

# Together AI (api.together.xyz → gratis con crédito inicial)
TOGETHER_API_KEY=xxx...

# Edge TTS (opcional, voces por defecto incluidas)
EDGE_TTS_VOICE_A=es-ES-AlvaroNeural
EDGE_TTS_VOICE_B=es-ES-ElviraNeural
```

### 3. Configurar base de datos Supabase

Ejecuta en Supabase → SQL Editor el schema completo (ver `/supabase/schema.sql` en el repo).

### 4. Ejecutar en desarrollo
```bash
npm run dev
```
