## ВҝQuГ© es EduAI Platform?

EduAI Platform es una plataforma educativa de siguiente generaciГіn que combina **25+ agentes de inteligencia artificial especializados**, un **Creator Hub con 8 formatos de salida potenciados por Gemini 2.5 Flash**, un **Audio Lab para transcripciГіn y ediciГіn de audio con IA**, un **Workspace de proyectos** para organizar todo tu material, y un **sistema de exГЎmenes para docentes con evaluaciГіn IA** para ofrecer una experiencia de aprendizaje completamente personalizada, adaptativa y gamificada.

Va mГЎs allГЎ de un simple chatbot вҖ” es un ecosistema inteligente donde mГәltiples agentes colaboran en paralelo para que el estudiante aprenda mГЎs rГЎpido y de forma mГЎs profunda. DiseГұada especialmente para el contexto educativo chileno, incluye cobertura curricular completa del **MINEDUC** (Parvularia, BГЎsica y Media) con escala de notas 1.0вҖ“7.0.

## InstalaciГіn y desarrollo

### Requisitos previos
- Node.js 18+
- Cuenta en [Supabase](https://supabase.com) (gratuita)

### 1. Clonar e instalar
```bash
git clone https://github.com/innova-space-edu/eduai-platform.git
cd eduai-platform
npm install
```

### 2. Variables de entorno
```bash
cp .env.example .env.local
```

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...
SUPABASE_SERVICE_ROLE_KEY=eyJxxx...

# IA вҖ” Gemini (requerido: Creator Hub, orquestador, transcripciГіn, evaluaciГіn exГЎmenes)
GEMINI_API_KEY=AIzaXxx...

# IA вҖ” Groq (requerido: streaming del tutor AGT)
GROQ_API_KEY=gsk_xxx...

# IA вҖ” OpenRouter (fallback general, opcional)
OPENROUTER_API_KEY=sk-or-xxx...

# ImГЎgenes вҖ” provider primario (opcional)
TOGETHER_API_KEY=xxx...

# ImГЎgenes вҖ” HuggingFace fallback adicional (opcional)
HF_TOKEN_1=hf_xxx...

# Cache y rate limiting вҖ” Upstash Redis (opcional, pero recomendado en producciГіn)
# Setup: Vercel Dashboard вҶ’ Storage вҶ’ Create вҶ’ Upstash KV вҶ’ Connect
UPSTASH_REDIS_REST_URL=https://...upstash.io
UPSTASH_REDIS_REST_TOKEN=...
```

> **Nota вҖ” voces del podcast:** Los hosts siempre se llaman **ГҒlvaro** (`es-ES-AlvaroNeural`) y **Elvira** (`es-ES-ElviraNeural`). Los nombres estГЎn definidos directamente en el cГіdigo del renderer y no pueden cambiarse por variables de entorno.

> **Nota sobre imГЎgenes:** Pollinations AI funciona **sin API key** вҖ” si no configuras `TOGETHER_API_KEY`, la generaciГіn de imГЎgenes seguirГЎ funcionando con FLUX gratuito vГӯa Pollinations.

> **Nota sobre Redis:** Si no configuras Upstash, la plataforma funciona igual вҖ” sin cache ni rate limiting. Para habilitarlo en Vercel, ve a *Storage вҶ’ Create вҶ’ Upstash KV вҶ’ Connect* y las variables se agregan automГЎticamente.

### 3. Supabase
- Ejecutar migraciones SQL en SQL Editor
- Crear buckets: `papers` (privado + RLS), `creations` (pГәblico), `chat-files` (pГәblico)
- Crear tablas `audio_transcriptions`, `projects` y `workspace_items` con RLS por usuario

### 4. Desarrollo
```bash
npm run dev
```
