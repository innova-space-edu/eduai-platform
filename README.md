## ВҝQuГ© es EduAI Platform?

EduAI Platform es una plataforma educativa de siguiente generaciГіn que combina **20+ agentes de inteligencia artificial especializados**, un **Creator Studio con 8 formatos de salida**, y un **sistema de exГЎmenes para docentes con evaluaciГіn IA** para ofrecer una experiencia de aprendizaje completamente personalizada, adaptativa y gamificada.

Va mГЎs allГЎ de un simple chatbot вҖ” es un ecosistema inteligente donde mГәltiples agentes colaboran para que el estudiante aprenda mГЎs rГЎpido y de forma mГЎs profunda. DiseГұada especialmente para el contexto educativo chileno, incluye cobertura curricular completa del **MINEDUC** (Parvularia, BГЎsica y Media) con escala de notas 1.0-7.0.

- **Sesiones de estudio sobre cualquier tema** вҖ” historia, ciencias, matemГЎticas, idiomas, y mГЎs
- **4 modos de aprendizaje:** Normal, SocrГЎtico, EvaluaciГіn y Colaborativo

### рҹ“Ӣ ExГЎmenes para Docentes

Sistema completo donde el docente crea pruebas con IA y las comparte mediante un link pГәblico:

```
DOCENTE                              ESTUDIANTE
в”Җв”Җв”Җв”Җв”Җв”Җв”Җ                              в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ

4. Publica вҶ’ obtiene link
5. Comparte por WhatsApp/Classroom    6. Abre link (SIN login)
                                      7. Ingresa nombre + curso + RUT
                                      8. Rinde con timer
                                      9. IA evalГәa desarrollo y V/F
                                     10. Ve nota + retroalimentaciГіn
11. Dashboard con 40 alumnos    вҶҗв”Җв”Җ resultados en tiempo real

**CaracterГӯsticas:**
- **3 tipos de preguntas:** Alternativas, Verdadero/Falso con justificaciГіn, Desarrollo
- **DistribuciГіn configurable:** el docente elige cuГЎntas de cada tipo
- **4 niveles de dificultad:** FГЎcil, Medio, DifГӯcil, Mixto
- **EvaluaciГіn IA con Gemini:** desarrollo y justificaciones evaluados con puntaje parcial y retroalimentaciГіn
- **Escala MINEDUC:** notas 1.0-7.0, exigencia configurable (50%-70%)
- **Link pГәblico sin login:** estudiante solo ingresa nombre, curso y RUT opcional
- **Timer:** 15-120 minutos con alertas visuales
- **LaTeX en preguntas:** fГіrmulas renderizadas correctamente
- **RetroalimentaciГіn completa:** cada pregunta con respuesta, correcta, y explicaciГіn
- **Dashboard docente:** tabla, distribuciГіn de notas, estadГӯsticas, auto-refresh 15s
- **Descarga:** Excel (.xlsx) con estadГӯsticas + PDF con resumen visual

### рҹ“„ Chat con Papers PDF

Arquitectura optimizada con Supabase Storage:

```

- Subida directa a bucket privado `papers` (hasta 50MB)
- ExtracciГіn: pdf-parse + fallback Gemini inline
- Chunking por relevancia segГәn la pregunta
- Seguridad RLS por usuario
- Soporte multilingГјe (ES/EN)

## InstalaciГіn y desarrollo

### Requisitos previos
- Node.js 18+
- Cuenta en [Supabase](https://supabase.com) (gratuita)

### 1. Clonar e instalar
```bash
git clone https://github.com/innova-space-edu/eduai-platform.git
cd eduai-platform
npm install
```

### 2. Variables de entorno
```bash
cp .env.example .env.local
```

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...
SUPABASE_SERVICE_ROLE_KEY=eyJxxx...

# IA (todas gratuitas)
GROQ_API_KEY=gsk_xxx...
GEMINI_API_KEY=AIzaXxx...
OPENROUTER_API_KEY=sk-or-xxx...

# ImГЎgenes
HF_TOKEN_1=hf_xxx...
TOGETHER_API_KEY=xxx...

# TTS (opcional)
EDGE_TTS_VOICE_A=es-ES-AlvaroNeural
EDGE_TTS_VOICE_B=es-ES-ElviraNeural
```

### 3. Supabase
- Ejecutar migraciones SQL en SQL Editor
- Crear buckets: `papers` (privado), `creations` (pГәblico), `chat-files` (pГәblico)

### 4. Desarrollo
```bash
npm run dev
```
